# Indomie Café

A responsive, touch-first mobile ordering experience for Indomie Café.

## Features

- Swipeable meal carousel with individual product artwork
- Meal details, quantity selection, and adjustable spice level
- Cart, promo codes, Spin to Win rewards, and two-step checkout
- Saved delivery addresses and payment selection
- Live multi-order tracking
- Editable profile, avatar, and notification preferences

## Run locally

1. Install Node.js 22 or newer.
2. Run `npm install`.
3. Run `npm run dev`.
4. Open [http://localhost:3000](http://localhost:3000).

## Production

Run `npm run build` to create an optimized Next.js production build.
